# Criterion Hooks

Report [hooks] for [Criterion].

## Dependencies

- Criterion [v2.3.3]

## Demo

- `make example-test` installs Criterion locally, compile the hooks, and example test.
- `make run-example` runs the example test with `-q -j1 --full-stats`.
